Запуск потоковой обработки

После того, как проект согласован и принято решение о его запуске, можно приступать к интеграции через API:

Шаг 1. Зарегистрируйтесь.
  1. Перейдите по ссылке и создайте аккаунт:
С помощью логина в Google

  1. В окне регистрации нажмите Google.

  2. Выберите нужный логин и подтвердите создание аккаунта.

С помощью электронной почты

  1. В окне регистрации введите адрес электронной почты и нажмите Log in.

  2. На вашу почту придет код подтверждения. Введите его в открывшемся окне.

  1. Завершите регистрацию, заполнив информацию о себе.

Шаг 2. Пополните баланс.

  1. Пополните баланс.

    1. Перейдите на вкладку Profile.

    2. Если у вас есть промокод воспользуйтесь им:

      1. Нажмите кнопку Enter promo code.

      2. В открывшемся окне введите промокод и нажмите кнопку Activate. После активации ваш баланс будет пополнен.

информация

Для дальнейшего использования вам потребуется привязать ваш аккаунт к биллинговой системе. Инструкцию по привязке аккаунта можно посмотреть по ссылке.

Шаг 3. Сообщите нам ваш логин.

Отправьте вашему консультанту логин, с которым вы зарегистрировались — мы привяжем к нему готовое решение для вашей задачи. Логин вы можете посмотреть в профиле.

Шаг 4. Дайте название проекту.

  1. Перейдите по ссылке, которую мы пришлем в ответ на письмо с логином.

  2. Создайте проект:

    1. В блоке Настройка проекта введите название приложения, затем нажмите Go to instructions setup.

    2. В блоке Настройка инструкции ничего вводить не надо, просто перейдите к следующему блоку — нажмите Go to Final check.

    3. В блоке Итоговая проверка нажмите Create project.

  3. На экране появятся данные созданного проекта. Скопируйте ID проекта и отправьте его вашему консультанту.

  4. Дождитесь активации проекта. Информацию о статусе вы увидите на той же странице, с которой копировали ID.

Шаг 5. Подключите API и начните пользоваться.

  1. Получите авторизационный токен: на вкладке Integrations вашего профиля нажмите кнопку Get OAuth token.

  2. Для обмена через API все готово — отправляйте данные для проекта и получайте результаты их модерации:

    • Используйте потоковую обработку данных (не пакетную). Инструкцию смотрите в Справке.

    • Вам понадобится ID проекта, полученный на предыдущем шаге. ID проекта в Справке соответствует переменная {app_project_id}.
